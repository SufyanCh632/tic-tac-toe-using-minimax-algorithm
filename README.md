# 🎮 Tic-Tac-Toe AI (Minimax Algorithm)

A simple command-line Tic-Tac-Toe game built in Python where you play against an **unbeatable AI** powered by the **Minimax algorithm**.

---

## 🚀 Features

* 🧠 AI uses Minimax (optimal decision-making)
* ❌ Human vs AI gameplay
* 🎯 Always plays the best move
* 📟 Simple CLI-based interface
* 🔁 Detects win, loss, and draw conditions

---

## 🛠️ Technologies Used

* Python 3
* Basic Data Structures (Lists)
* Recursion (Minimax Algorithm)

## 🎯 How to Play

* The board positions are numbered from **1 to 9**:

```
| 1 | 2 | 3 |
| 4 | 5 | 6 |
| 7 | 8 | 9 |
```

* Enter a number (1–9) to place your move.
* You play as **X**
* AI plays as **O**

---

## 🧠 How AI Works

The AI uses the **Minimax Algorithm**, which:

* Simulates all possible moves
* Chooses the optimal move
* Maximizes its chances of winning
* Minimizes the opponent’s chances

### Evaluation System:

* `+1` → AI wins
* `-1` → Player wins
* `0` → Draw
